## 解法一

第一种解决方法是使用递归。这是经典的方法，直截了当。我们可以定义一个辅助函数来实现递归。
```javascript

var preorderTraversal = function(root) {
    let arr = []
    help(root,arr)
    return arr
};
function help(root,res){
    if(root!==null){
        res.push(root.val)
        if(root.left!==null){
            help(root.left,res)
        }
        if(root.right!==null){
            help(root.right,res)
        }
    }
}

```

复杂度分析

时间复杂度：O(n)。递归函数 T(n) = 2⋅T(n/2)+1。
空间复杂度：最坏情况下需要空间O(n)O，平均情况为O(logn)。
